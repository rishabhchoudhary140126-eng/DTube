Project Status

I started learning backend development roughly two months ago. At the time of this submission I had not yet learned MongoDB, JWT, sessions, or OAuth. Rather than postpone the project or copy implementations I couldn't explain, I chose to build the platform using the concepts I understood—Express, middleware, JSON storage, and Multer.

This project is intentionally incomplete with respect to the induction specification. My goal was to demonstrate my problem-solving approach, willingness to learn, and ability to build a functional system from first principles. As I continue learning, the file-based storage and custom authentication middleware will be replaced with proper database-backed authentication and JWT-based authorization.
